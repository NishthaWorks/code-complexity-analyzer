# 💻 Code Complexity Analyzer

A smart tool to analyze time complexity of code using pattern detection.

## 🚀 Features
- Detects loops (for, while)
- Identifies nested loops (O(n^2), O(n^3))
- Recursion detection
- Binary search detection (O(log n))
- Provides optimization suggestions
- Built with Streamlit UI

## 🛠️ Tech Used
- Python
- Streamlit

## ▶️ How to Run
```bash
streamlit run app.py
